# 🍽️ Culinary Brand Builder

A premium, modern Food Consultancy website built with **React**, **Vite**, **TypeScript**, and **Tailwind CSS**. The website features a luxury UI/UX, responsive design, smooth animations, and a professional brand identity for culinary consulting businesses.

## ✨ Features

- Modern & Luxury UI/UX
- Fully Responsive Design
- Smooth Framer Motion Animations
- Premium Typography
- Portfolio / Case Studies
- Services Section
- About Section
- Contact Form
- SEO-Friendly Structure
- Fast Performance
- Mobile First Design

## 🛠️ Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion
- React Router
- React Hook Form
- Zod
- Lucide React

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/sakshipatel17/culinary-brand-builder.git
```

Go to the project directory:

```bash
cd culinary-brand-builder
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
